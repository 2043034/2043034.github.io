Tools

This is a compilation of a bunch of different web tools I have written.

AESCrypt - AESCrypt is a tool that allows you to encrypt messages with the AES military-grade encryption algorithm.
STE - STE (Simple Text Encryption) is a little tool that allows you to encrypt text with a simple symmetric algorithm I designed. It's not military-grade, but will hold up under casual observation.
